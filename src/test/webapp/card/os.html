Chance (p.58)
  • Repousser les limites : Ajouter l’indice de CHC au test, avant ou après le lancer. Appliquer "Règle des Six" et "No Limits"
  • Seconde chance : Relancer tous les dés en échec. N’annule pas les complications, ou échecs critiques.
  • Prendre l’initiative : Personnage joue en premier quel que soit son score, pour un tour de combat.
  • Blitz : Lancer 5 dés d’initative pour un tour de combat.
  • De justesse : Annuler les effets d’une complication. Changer un échec critique en simple complication.
  • Ultime action : {CON + VOL} (3) SI mis hors combat, le personnage peut faire une dernière unique action (tout type).
Leadership (p.144)
{Leadership + CHA} [Sociale]
Action complexe
Commandement
  • Résistance : Leadership + Volonté [Sociale]
  • La cible accepte le personnage commme leader pour un tour de combat par succès excédentaire.
Inspiration
Les succès agissent comme un test d'éqauipe sur pour un subordonné.
Ralliement
Tous les subordonnés ajoutent 1 à leur score d'initiative pour chaque 2 succès.

Dégâts*
VD + Succès excédentaires de l’Attaque
Armure*
Armure - PA
Type de dégâts
  • Armure* - Dégâts* >= 0 :
        Dégâts physiques
  • Armure* - Dégâts* < 0 :
        Dégâts étourdissants
  • Certaines armes n'infligent que des dégâts étourdissants
  • Lorsque le moniteur de condition étourdissant est plein, tous les dégâts sont physiques.
